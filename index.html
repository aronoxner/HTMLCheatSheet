HTML Reference Guide

by Aron Ochsner

Basic Tags

Tag Name Description
<html> HTML Creates an HTML document
<head> Head Sets off the title and other info that isn't displayed
<body> Body Sets off the visible portion of the document
<title> Title Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked

Formatting

Tag Name Description
<p> Paragraph Creates a new paragraph
<br> Line Break Inserts a line break (carriage return)
<blockquote> Block Quote Puts content in a quote -indents text from both sides
<div> Division Used to format block content with CSS
<span> Span Used to format inline content with CSS

Text Tags

Tag Name Description
<pre> Preformatted Text Creates preformatted text
<h1> -> <h6> Section Heading Creates headlings -- H1 = largest, H6 = smallest
<strong> Strong Importance Creates bold text
<em> Emphasis Creates italicized text
<code> Inline Code Used to indicate text is computer code, usually uses monospace font
<cite> Citation Creates a citation, usually in italics
<address> Contact Address Creates address section, usually in italics

Lists

Tag Name Description
<ul> Unordered List Creates and unordered list
<ol> Ordered List Creates an ordered list
<li> List Item Encompasses each list item. Must be contained in a parent ordered list, unordered list, or menu
<dl> Description List Creates a description list
<dt> Description Term Specifies a term in a definition or description list
<dd> Description Details Provides a definition or related text associated with a Description Term

Graphical Elements

Tag Name Description
<img src="URL> Image Adds image, source attribute required
